Latar Belakang Masalah

Mengapa sistem ini dibutuhkan?

Pengelolaan BBM operasional Polda Jawa Barat menghadapi rantai permasalahan yang saling terkait akibat ketergantungan penuh pada proses manual.

01 · PEMBUATAN KUPON
Input Manual 764 Kendaraan per Bulan
Data tiap kendaraan harus diinput satu per satu di Excel — nomor kupon, jenis BBM, penandaan hari libur — tanpa otomatisasi. Tidak ada validasi masa berlaku kupon secara digital.
02 · PENCATATAN TRANSAKSI
Rekapitulasi Tertunda 1–3 Hari
Tiga petugas merekap ~1.000 transaksi/bulan ke lembar Excel 60+ kolom. Keterlambatan input menyebabkan data tidak mutakhir dan memperlemah kemampuan verifikasi real-time.
03 · VERIFIKASI KUOTA
Rentan Kupon Duplikat
Tanpa identifikasi unik digital, kupon yang kuotanya habis bisa digunakan ulang karena fisik kupon tidak berubah. Pengawasan hanya bersifat korektif pasca-audit.
04 · MONITORING STOK
Keputusan Restok Berbasis Data Usang
Tidak ada sensor atau sistem monitoring tangki otomatis. Stok opname fisik 20–30 menit hanya dilakukan 2–3 kali per bulan, menciptakan jeda pengawasan yang kritis.
05 · PELAPORAN
Laporan Harian Tidak Terpenuhi
SOP mewajibkan laporan harian, mingguan, bulanan, dan tahunan. Namun keterlambatan rekapitulasi membuat laporan harian praktis tidak dapat disajikan tepat waktu secara konsisten.
06 · ANALITIK ALOKASI
Tidak Ada Basis Analisis Alokasi
Fenomena kupon minus berulang menandakan kesenjangan alokasi vs kebutuhan aktual, namun tidak ada sistem yang dapat mengidentifikasi pola dan memberikan rekomendasi penyesuaian.
Fitur Utama Aplikasi

Satu platform untuk seluruh
siklus pengelolaan BBM

SIGMABAR mencakup empat modul terintegrasi yang menjawab setiap titik permasalahan dalam alur distribusi dan transaksi BBM Polda Jawa Barat.

Pengelolaan Kupon Digital
Otomatisasi pembuatan kupon BBM siap cetak berbasis data master kendaraan, lengkap dengan QR code unik per kupon, penandaan hari libur otomatis, dan validasi masa berlaku sesuai Peraturan Aslog No. 1/2023.
Kupon & QR Code
Pencatatan Transaksi Real-Time
Aplikasi mobile Android memungkinkan petugas Pom Polda mencatat transaksi secara digital pada saat pengisian berlangsung. Data langsung tersinkronisasi ke desktop tanpa perlu input ulang.
Mobile · Near Real-Time
Dashboard Monitoring & Analitik
Visualisasi KPI penyerapan kuota per SATKER, tren penggunaan BBM, estimasi sisa stok tangki, dan deteksi anomali kupon minus — memberikan dasar keputusan yang kuat bagi KASUBAG Biro Logistik.
Analitik Deskriptif
Pelaporan Otomatis Multi-Periode
Laporan harian, mingguan, bulanan, dan tahunan dihasilkan secara otomatis dari database terpusat sesuai format blanko SOP Biro Logistik, tanpa pengolahan manual dari Excel.
Harian · Bulanan · Tahunan
Arsitektur Serverless Lokal
Dirancang tanpa server terpusat — 1 desktop sebagai database utama dan 3 mobile sebagai terminal input tersinkronisasi via jaringan lokal, memastikan data terbaru tersedia tanpa infrastruktur cloud.
Flutter · Android · SQLite
Rekomendasi Alokasi BBM
Analisis kesenjangan antara kuota yang dialokasikan dengan konsumsi aktual per SATKER, mengidentifikasi pola kupon minus berulang sebagai dasar penyesuaian alokasi periode berikutnya.

Arsitektur Teknis

Dirancang untuk keterbatasan infrastruktur

Tidak ada server terpusat yang tersedia — namun bukan hambatan. SIGMABAR beroperasi penuh dengan arsitektur serverless berbasis sinkronisasi jaringan lokal.

ARSITEKTUR SISTEM — SERVERLESS LOCAL SYNC
Desktop App (Flutter)
Database Utama · Master Data · Laporan & Dashboard
Sync via Local Network
Mobile App × 3 (Android)
Input Transaksi · Scan QR · Verifikasi Kupon
Export / Print
Output — Laporan & Kupon
Format Blanko SOP · Excel Export · PDF Kupon Cetak
  • Flutter (Desktop & Android)
    Codebase tunggal untuk desktop dan mobile memastikan konsistensi logika bisnis dan mempermudah pemeliharaan jangka panjang.
  • SQLite — Penyimpanan Lokal
    Database ringan tanpa dependensi server. Data transaksi, kupon, dan kuota tersimpan persisten di perangkat desktop sebagai node utama.
  • QR Code Identification
    Setiap kupon mendapat kode unik di-encode dalam QR code. Scan via kamera mobile menvalidasi identitas dan sisa kuota secara instan.
  • Analitik Deskriptif
    Agregasi statistik berbasis data historis menghasilkan persentase penyerapan kuota, frekuensi kupon minus, dan pola fluktuasi penggunaan per SATKER.
  • Import / Export Excel
    Kompatibel dengan format data master Polda Jawa Barat dan format blanko laporan SOP Biro Logistik untuk kesinambungan dengan sistem eksisting.
Pemangku Kepentingan

Siapa yang mendapat manfaat?

SIGMABAR dirancang untuk memenuhi kebutuhan seluruh aktor dalam ekosistem pengelolaan BBM Polda Jawa Barat.

Kepala Subbag SATKER Biro Logistik
Mendapatkan dashboard analitik lengkap untuk pengambilan keputusan belanja BBM yang lebih tepat waktu, akuntabel, dan berbasis data aktual — bukan estimasi manual.
Staff Kantor SATKER Biro Logistik
Beban kerja rekapitulasi manual berkurang drastis. Pembuatan kupon, rekap transaksi, dan penyusunan laporan dilakukan lebih cepat dan akurat melalui sistem terotomatisasi.
Petugas Pom Bensin Polda Jawa Barat
Verifikasi kupon melalui scan QR code via mobile, menggantikan pengecekan visual manual yang rentan terhadap kupon duplikat dan data kuota yang tidak mutakhir.
Supir / Pemegang Kupon Kendaraan Dinas
Mendapat kepastian terkait sisa kuota dan status kupon secara digital, mengurangi ketidakpastian yang sebelumnya hanya diketahui melalui catatan fisik di kertas.
